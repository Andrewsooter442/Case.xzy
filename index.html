Snake Playing Algorithm

I made this project a while back after getting inspiration from a video from AlphaPhoenix.
The path-finding algorithm is not that complex; to be honest, the snake always follows a Hamiltonian path, and whenever it can cut the board in such a way that it can return to the original Hamiltonian path, it will cut the board and save some moves.
However the rendering of snake uses logic so complex that I could have solved world hunger if I was not writing it.

Prey-Predator Simulation

I made this project as part of WOC organised by SDSLabs.
It is a Python-based simulation where you can tweak the initial conditions and other parameters of an enviroment and then prey-predator adapt to each other and try to survive for a long time. (You are not playing GOD!)
